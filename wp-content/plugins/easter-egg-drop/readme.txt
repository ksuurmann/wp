A simple plugin show an easter egg dropping in from the top of the screen


== Description ==

This plugin makes an easter egg drop from the top of the screen after a random delay of between 1-5 seconds.  

When clicked it triggers an alert box with a customisable message.


== Installation ==

Install Easter Egg Drop like you would install any other WordPress plugin.

Dashboard Method:

1. Login to your WordPress admin and go to Plugins -> Add New
2. Type "Easter Egg Drop" in the search bar and select this plugin
3. Click "Install", and then "Activate Plugin"


Upload Method:

1. Unzip the plugin and upload the "easter-egg-drop" folder to your 'wp-content/plugins' directory
2. Activate the plugin through the Plugins menu in WordPress
